# **LemonAPC**
---
## ***HOW TO INSTALL***
### **JavaScript**
- Import via Import
```javascript
let lemonAPC;
import('url/to/lemonAPC')
   .then(module => {
      lemonAPC = module.lemonAPC;
   });
```
- That's all
---
## ***NOTE***
- A theme is **required** for it to work.
